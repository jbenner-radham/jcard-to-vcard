# jcard-to-vcard

A jCard to vCard converter. Currently CLI only but will have a programmatic
module interface in the future.

## Heads Up!

This is still very much a WIP project.

## Usage

```sh
node dist/cli.js my-jcard.json > my-new-vcard.vcf
```
